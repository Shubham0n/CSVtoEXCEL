## Dependencies

python 3

## Installation

1. CD CSVtoEXCEL

2. Create venv
```
> Mac Os
python3 -m venv env

> Window
py -m venv env
```

3. Active env

```
> Mac Os
source env/bin/activate

> Window
env/Scripts/activate
```

4. Install the required packages:

```
pip install -r requirements.txt
```

5. Run script:

```
python3 csv_to_excel.py
```
